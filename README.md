## 📥 Installation Instructions
1. Clone the repository: `git clone https://github.com/ALIMOHAMMEDABUBAKAR/my-portfolio.git`
2. Navigate to the project directory: `cd my-portfolio`
3. Open `index.html` in your preferred web browser.

## 🛠 Usage Guide
Use the navigation bar to explore various sections of my portfolio, including my skills, projects, and testimonials. You can also toggle between light and dark modes.

## 🚀 Deployment Methods
This portfolio can be easily deployed using GitHub Pages:
1. Push your code to a GitHub repository.
2. Go to your repository settings.
3. In the 'GitHub Pages' section, select the `main` branch as your source.

## 🎨 Customization Guide
- **Content:** Modify the `index.html` file to update personal information, skills, projects, and testimonials.
- **Styling:** Adjust the `css/style.css` file, utilizing the defined CSS variables for consistent theming.
- **Functionality:** Enhance or modify interactive elements in `js/script.js`.

## 🧹 Code Formatting and Linting
To maintain code quality and consistency, this project uses Prettier for code formatting and ESLint for linting. 

### Setup
1. Install Node.js and npm (if not already installed).
2. Install Prettier and ESLint as development dependencies:
   ```bash
   npm install --save-dev prettier eslint
   ```
3. Create `.prettierrc.json` and `.eslintrc.json` configuration files in the project root.

### Usage
- **Format code:** `npx prettier --write .`
- **Lint code:** `npx eslint .`
