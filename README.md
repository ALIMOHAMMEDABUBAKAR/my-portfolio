## рҹ§№ Code Formatting and Linting
To maintain code quality and consistency, this project uses Prettier for code formatting and ESLint for linting. 

### Setup
1. Install Node.js and npm (if not already installed).
2. Install Prettier and ESLint as development dependencies:
   ```bash
   npm install --save-dev prettier eslint
   ```
3. Create `.prettierrc.json` and `.eslintrc.json` configuration files in the project root.

### Usage
- **Format code:** `npx prettier --write .`
- **Lint code:** `npx eslint .`
